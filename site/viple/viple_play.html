Other games to help learn Viple

  • Vim Adventure
  • Vim Racer
  • PacVim

Level 1: Puffy Fish!

Pass seven obstacles to advance to next level.

J -- Up
K -- Down
C -- Skip level

Level 2: Bricker!

Clear the bricks to advance to the next level.

H -- Left
L -- Right
C -- Skip level

Level 3: Snake!

Guide the snake to eat the apples to grow the snake.

H -- Left
L -- Right
C -- Skip level

Level 4: Insert Mode!

Enter Insert Mode to eat the apple. Exit Insert Mode to turn the snake.

I -- Enter Insert Mode
Esc -- Exit Insert Mode

Level 5: Connect Three!

Delete lines to connect 3 matching jewels. Matching gems will turn the squares gold. Turn all squares gold to advance to the next level.

J -- Down
K -- Up
D, D -- Delete line
D, [2, 3, 4, ...], Enter -- Delete multiple lines

Level 6: Visual Mode!

Enter Visual Mode to select gems to delete. Turn all squares gold to advance to the next level.

H, J, K, L -- Left, Down, Up, Right
V -- Enter Visual Mode
D -- Delete selection
Esc -- Exit Visual Mode

Level 7: Challenge Game!

Use the skills you have learned to turn all squares to gold.

H, J, K, L -- Left, Down, Up, Right
V -- Enter Visual Mode
I -- Enter Insert Mode
Esc -- Exit Visual/Insert Mode
D -- Delete selection
D, D -- Delete line
D, [2, 3, 4, ...], Enter -- Delete multiple lines
